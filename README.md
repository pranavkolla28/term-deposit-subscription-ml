\# Term Deposit Subscription ML



Machine learning project to predict customer subscription to term deposits using Portuguese bank marketing data.



\## Problem

Banks run marketing campaigns to promote term deposits. This project predicts whether a customer will subscribe.



\## Approach

\- Exploratory Data Analysis (EDA)

\- Handling class imbalance using SMOTE

\- Random Forest model

\- Evaluation focused on recall



\## Tools

\- Python

\- Pandas, NumPy

\- Scikit-learn

\- Matplotlib, Seaborn



\## Outcome

Improved identification of potential subscribers and provided insights to improve marketing effectiveness.
